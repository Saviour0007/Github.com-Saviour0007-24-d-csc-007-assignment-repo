## 🛠 Setup Instructions

1. Install XAMPP (or any PHP + MySQL environment).
2. Start *Apache* and *MySQL* in XAMPP.
3. Open *phpMyAdmin* (http://localhost/phpmyadmin).
4. Create a new database (e.g. student_db).
5. Import the provided student_db.sql file into that database.
6. Place all the PHP files in the htdocs folder of XAMPP (e.g. C:\xampp\htdocs\student_registration).
7. Open your browser and go to:  http://localhost/csc-24-d-csc-010-assignment-repo/register.php
8. You should now see the application.
